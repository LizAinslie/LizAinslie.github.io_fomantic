Caboose
Abandoned (2019-2020)
Caboose was a dynamically-typed programming language built on a powerful VM. I may consider revisiting it using a different language someday.
GitHub
AlpineOS
WIP (2021-current)
AlpineOS is meant to be a fresh take on modern computing. The goal here is to be as UI fluent as MacOS with compatibility to run Linux executables and possibly Windows executables, as well as its own executables. I currently only work on this when I need something to do badly, because I really don't wanmt to mess with EFI.
